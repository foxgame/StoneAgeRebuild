# 游戏多开
- 玩过石器的人都知道，这游戏不多开没法玩。
- 如何多开：PC默认支持多开。Mac 端直接复制图标。
- 安卓版本，支持云多开，例如"红手指远程"等。
- 游戏设置，PC默认1280*800，支持最小化，Windows系统可以把游戏缩小至最小（非常省资源）

# 建立人物
- 骑雷龙防高，人物PK不容易死，建议选择高防雷龙。老虎敏高容易躲闪，建议老虎雷龙都要准备。
- 选择好你想骑得宠就可以确定人物了。

- 这里我们先来一份自强的攻略，几乎不靠其他人。首先，一定要取一个响亮的名字。这里不得不提一下我和朋友见过最厉害的名字：黄贱滑

## 加点
- 体力：血量
- 腕力；攻击
- 耐力；防御
- 速度；敏捷
- 新人先加敏捷 70-100，剩下的全加攻。简单说这样既方便练级又方便做任务
（题外话，合击的条件是我方敏捷均高于敌人，或者均低于敌人。理解这点对大家将来练宠，练级都很有帮助）
不用改点酒三转考虑改点，用改点酒直接 5 转改点（改点酒需要来吉卡购买，均价1元左右）。

## 属性：
- 地克水
- 水克火
- 火克风
- 风克地。
- 现阶段建议55属性，人物要提高存活率。其实你想什么属性也都行。

## 完成建立人物后
- 纯新手请熟悉各个按钮，请点击左上角 menu，打开组队，交易，显示名称。
- 官网地址点击系统
