# 石器时代重制版！sqfox原版重制!十周年纪念！
## Unity完全重制，支持全平台，支持Mod，插件脚本、Lua脚本，专属服务器等，原汁原味，被玩家评价为还原度最高的石器时代！

## 游戏特色
- 石器重制版一区，使用3.0原版模板，部分4.0功能，四大家族，其他功能随着游戏时间逐步开放。
- 专属自建服，可使用3.0原版模板，5.0原版模板，根据需求自行修改。
- 永久开放，石器养老服！

- 本服开发目的使用老游戏为模板，进而开发通用2D、3DMMORPG开发平台。具体详见更新Timeline。所有模板资源来自互联网版权属于版权方，代码均为原创版权属于MMORPG开发组。
