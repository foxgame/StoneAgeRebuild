# Chat指令

## 命令,格式,权限,描述

### 客户端

- /help0,,0,查询帮助
- /fame,空格 数量,0,兑换声望
- /fightf,,0,家族战强制战斗
- /team,空格 (1-4),0,踢出队伍
- /lover,空格 ID,0,绑定爱人
- /bet,0,空格 数量,赌场下注
- /name,空格 新名字,0,改变名字，需要消耗点卷
- /a,空格 指令描述，需要可操作当前角色、其他角色、游客等，需要绑定大模型API Key，可自行在deepseek等平台申请。

## 服务器

- /warp,空格 地图ID 坐标x 坐标y,200,跳转地图
- /item,空格 角色ID 道具ID 数量,999,添加道具
- /player,空格 角色名称,400,查询玩家信息
- /pet,空格 角色ID 宠物ID 成长(可选),999,添加宠物，成长不会低于最低成长，不会高于最高成长+1。
- /petlv,空格 角色ID 宠物ID 等级,600,升级宠物
- /baseexp,空格 经验值倍率,600,经验值倍率
- /basefame,空格 声望倍率,600,声望倍率
- /npcrandwarp,空格 0/1，关闭开始npc随机位置。
- /cchatban,空格 角色ID,200,取消禁止聊天
- /chatban,空格 角色ID,200,禁止聊天
- /chatbantext,空格 敏感词,200,禁止敏感词语
- /cauctionban,空格 角色ID,200,取消禁止拍卖行
- /auctionban,空格 角色ID,200,禁止拍卖行
- /cbanf,空格 家族ID,200,取消禁止家族
- /banf,空格 家族ID,200,禁止家族
- /cban,空格 角色ID,200,取消禁止玩家
- /ban,空格 角色ID,200,禁止玩家
- /clearban0,,200,清空禁止玩家列表
- /clearban1,,200,清空禁止家族列表
- /clearban2,,200,清空禁止拍卖行列表
- /clearban3,,200,清空禁止聊天列表
- /clearban4,,200,清空禁止敏感字列表
- /kick,空格 角色ID,200,踢出玩家
- /pay,空格 角色ID 点卷数量 实际金额,充值点卷
- /guest create,空格 游客id 颜色 坐骑,创建游客
- /guest move,空格 地图id 东 南 游客id,移动游客
- /guest remove,空格 游客id ,删除游客
- /guest join,空格 游客id1 游客id2,组队游客
- /mod,空格+地址,400,加载服务器MOD
- /reload,,999,重新加载配置文件
- /exit,,999,重启服务器
